About Me

I am a Senior Software Engineer at Couchbase Inc.

My work targets the storage engines in Big Data Management Systems. In particular, I work on reducing storage size to accelerate scan-based analytical workloads for document store systems.

Research Projects

LSM-based Tuple Compaction Framework

Proposed a new mechanism to leverage Log-Structured Merge (LSM) lifecycle events to automatically infer schema and semantically compact self-describing semi-structured records. Implemented in Apache AsterixDB along with page-level compression, achieving a 9.8x reduction in storage size and query speedups of the same factor.

Extended Paper

Columnar Formats for Schemaless LSM Document Stores

Extended the Dremel columnar format to support document stores' flexible, nested data model. Designed and implemented two layouts for organizing and storing schemaless data in LSM-based storage systems, optimizing query scans over nested, sparse document fields.

Extended Paper

Code Generation for Schema-on-Read Databases

Explored runtime query compilation techniques for document stores where value types are unknown until query time. Leveraged Oracle Truffle to compile execution sub-trees in a Java-based document store, showing major speedups over standard vectorized batch processing models.
